## GnuPG
---
####如果你采用的是GnuPG的gui界面来生成密钥并用于git，大概率要完成以下事情：

::: Notice
找到gpg.exe路径，一般位于GnuPG文件夹的bin路径下，例如"D:\GnuPG\bin\gpg.exe"
:::
::: details 点我查看代码 
```注意
git config --global gpg.program "D:\GnuPG\bin\gpg.exe"
``` 
:::
